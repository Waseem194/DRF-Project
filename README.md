# DRF-Project

## 🚀 How to Run This Project

1. Make sure you have Python 3.14.3 installed.
2. Create virtual envirement Through this command: "python -m venv .venv".
3. After create virtual envirement Then install requirements.txt through this command: "pip install -r requirements.txt".
4. After this to Navigate manage.py file then run this command: "python manage.py runserver".
